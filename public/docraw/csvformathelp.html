Each CSV row is similar to an API Call

[asset] determines the type of element(s) created

Unused columns are ignored, allowing all columns to be included with all rows, as needed


Importing CSV Data into frames - CSV record types
CSV column/field names are all lower case, and when possible convert (with case changes) directly to the field name on the babylon/webGL asset.

materialname
x (positionX)
y (positionY)
z (positionZ)

ry (rotationY)
rz (rotationZ)
sx (scalingX)
