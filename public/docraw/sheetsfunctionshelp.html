Custom Sheets Functions

=getDataRangesForSheet(sheetName)

Heavily Used Function

This function returns a list of ranges containing the data tables for a sheet.
- data must start in the first (A) column
- first row is headers
- each row must have an entry in the first column, or else it's an end of table
- the data table/range is ended when the first blank row shows up (no entry in first column)
- rows with no data in the first column are ingnored, allowing for comments
