Camera And Lighting

Assets and blocks without a camera specificed use the Arc Rotate Camera.
- zoom in and out with the mouse wheel, page up/dn or gestures
- right mouse button to move the point of focus
- left mouse allows you to rotate around the point of focus

Blocks without a light specificed use a Hemispheric Light with color (0.2, 0.8, 0.2)
- intensity is adjustable - 1.0 is approximately sunlight.
- adding a light to a block (or any child block), the default light is disabled.

Blocks specify a skybox and ground when acting as a scene (root - not linked child).

Shortcuts and macros

Materials and Textures: rgb (0-1 levels) color, use the format "color:r,g,b" - i.e. "color:1,0,0" is diffuse red, for emissive color, use "ecolor:1,0,0", to set both use "decolor:1,0,0"
  for textures there is an image library available using the prefix "sb: 123.jpg"

Meshes: there is an babylon file library available using the prefix "sb: mesh.babylon".

Frames: cp (clone previous), lf (last frame) generate redundant frames
  %, +, ++ and * for computed time, position, color, scale and rotate

Blocks: Tools for generating scenes, shapes, 3D text, connector and animated lines

Workspace Generation: Tools for generating virtual stores from CSV based data
