BumSleeper PWA Applications

A bed and bath listing service for local missions and churches to offer and reserve beds
BumChat - so bums can leave messages for each other
Notifications - so a bum can wake up and come inside if a bed opens up
BumRatings - get ranked better if well behaved
BedRate - Bums rank the beds and comment
BathRate - good plumbing is a right
PublicFeed - free for all posting - with iggies (ignores)
Groups and Roles - owners, managers and users for beds
