Cloud Functions (REST API)

Swagger is used as a framework inside the cold boot node servers. Cloud functions are a "serverless" backend - meaning they don't run or cost when not used, but scale immediately on demand (and you pay for what you use).

Google Cloud Platform (GCP)

GCP is used to fill in the gaps where Firebase doesn't fill all the needs. GCP is a fully featured cloud platform - and tightly integrated with Firebase.
