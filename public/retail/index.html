How to compare visual catalogs to similar software


Virtual Conference software is offered by a number of companies that includes shared audio and 3D virtual experiences, with and without a VR headset. Apps and catalogs are what you can do in a conference.

Using device orientation the right way


Device orientation is common on most phones and used mostly to swap between portrait and landscape screen - and occassionaly as a compass - a decent amount in games for steering.
If used in a limited range, device orientation can be used as UI feature to show additional screen space - but care must be taken to work well with minimum physical motion.
