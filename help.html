Postfix (offset) Operators
            i.e. 10ms, 20s, 30%, 10, 20deg
        + Operator: frame starts after the previous frame
            i.e. 50ms+, 100s+, 1%+, 10deg+, 10+

        cp Operator: Clone Previous frame X milliseconds before current frame (for snappy transition)
            i.e. 1200cp250 1000ms++cp100 or 10scp400
        lf Operator (Last Frame - clone this as last frame - so element doesn't slide) - cplf supported

    Times can be in ms (millisecond, default), s (second), m (minute) or % values
        i.e. 0, 10ms, 10s, 10m, 10%, 10ms++ (note the offset operator is last)

    % Offsets
        require specified run length
